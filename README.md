# Ticket Vendor Machine - Assignment 1 (SE)


## Project Deliverables
* **Q&A_Requirements.xlsx**: A questionnaire clarifying ambiguous or missing requirements of the initial system.
* **Design Document **:
    * Use Case Diagram & Description 
    * Activity Diagram 
    * Sequence Diagram 
    * State Chart Diagram
    * Class Diagram (Software architecture and class structures).
* **UI Wireframes/Mockups**: Screen designs of the interface built with Figma 


## Key System Features
* Selection of routes, ticket quantities, and passenger types (Adult, Child, Student).
* Multiple payment methods: Cash (with change return), Credit Card, and QR Code via digital wallets.
